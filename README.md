## About The Project

[![Product Name Screen Shot][product-screenshot]](https://armck-hub.github.io/Demo-To-Me/)

Demo-To-Me is a stand-alone, static Web Form, giving you the ability to test out the Mail-To-Me application! Feel free to input your receiving email address and watch your message arrive within seconds. Please use responsibly.

Here's why Demo-To-Me is important:
* Allows for you to test out our Mail-To-Me application – Easy!


### Built With
* HTML, CSS, BootStrap
